# GeoFS FIP Flight Information Plugin v1.2
## Update Release Notes ✈️

---

### 🎯 Update Overview
This v1.2 update delivers **polished UI consistency, expanded logging capabilities, and refined usability**, while preserving 100% of the original plugin logic, layout, and core functionality. It’s built to make your GeoFS flight logging smoother, more complete, and visually consistent.

---

### ✨ What's New in v1.2
#### 1.Unified Dark Theme UI
- Full-panel **white text on black background** standardization across all elements:
  - Buttons, labels, input fields, and status text all follow a consistent dark theme
  - High visibility in all GeoFS flight scenes, reduced eye strain during long-haul flights
  - Zero changes to original panel structure, so your workflow stays identical

#### 2.New "Flight Date" Logging Field
- Added a dedicated **Date** input field *before* the flight number, to log your flight’s execution date
- Complete multi-language support: Fully localized for Simplified Chinese, English, and Traditional Chinese
- Perfect for airline simulation, virtual airline operations, and detailed flight log management

#### 3.Usability & Layout Polishes
- **Reset Button Relocation**: Moved the `Reset` button to the left of `Close` for more intuitive, one-handed operation
- **Default Language Update**: Set **English** as the default language for global users, with full multi-language switching retained
- **Cleaner Interface**: Removed redundant "Optional" placeholder text from all language versions for a streamlined look

#### 4.Experience & Compatibility Enhancements
- 100% retention of all core features: Flight timer, K-key toggle, floating button control, input conflict fix, etc.
- One-click Tampermonkey installation: Compatible with all major browsers, no extra configuration required
- Input conflict protection: Automatically disables game hotkeys when typing in input fields, eliminating accidental in-flight controls

---

### Full Feature List
✅ **Flight Information Panel**: Log flight date, flight number, aircraft type, departure/arrival airports, and estimated flight time

✅ **Flight Timer**: One-click start/stop for accurate flight duration tracking

✅ **Multi-language Support**: Seamless switching between Simplified Chinese, English, and Traditional Chinese

✅ **Quick Panel Toggle**: Dual control via bottom-left "Geofs FIP" floating button and K-key hotkey

✅ **Floating Button Visibility Control**: Hide/show the bottom-left button directly from the panel for an unobstructed view

✅ **Input Conflict Prevention**: No game control triggers when typing in input fields for smooth data entry

✅ **One-click Reset**: Instantly clear all flight data to start a new log

✅ **One-click Tampermonkey Installation**: Full browser compatibility, ready to use out of the box

---

###  Installation & Usage
1.  Copy the full plugin code, create a new script in Tampermonkey, and paste the code
2.  Save the script and refresh the GeoFS page; the plugin will load automatically
3.  Open the panel via the bottom-left "Geofs FIP" button or the K-key, fill in your flight details, and start the timer to begin logging
