GeoFS FIP Flight Information Plugin v1.4
Update Release Notes ✈️
Update Overview
This v1.4 update focuses on UI refinement and branding consistency, perfecting the bottom bar toggle button to match GeoFS native styling 1:1, while keeping all core functionality fully intact. It delivers a seamless, integrated cockpit experience for every virtual pilot.
✨ What's New in v1.4
Branded Bottom Bar Toggle Button
Renamed button to Geofs FIP for consistent, accurate branding
100% visual match to native GeoFS UI elements (font size, weight, alignment, spacing)
One-click panel toggle directly from the in-game bottom bar
Smooth native-style hover effects and interaction
Precision UI Alignment
Fixed text centering for the bottom bar button, eliminating offset
Refined button sizing to perfectly match adjacent native controls
No layout shifts, no visual clutter, fully integrated with the cockpit
Core Functionality Fully Preserved
Real-time flight timer and status tracking
Full flight detail logging (flight number, aircraft, route, ETE)
Save & view mode for clean flight detail display
Hotkey K for quick panel toggle
Full multi-language support (English / 简体中文 / 繁體中文)
Hideable button for flexible UI control
🛠️ Installation
Install a userscript manager (Tampermonkey, Violentmonkey, etc.)
Click Install on the script page
Launch GeoFS — the FIP panel will load automatically in the top-left corner
Use the bottom bar Geofs FIP button or hotkey K to toggle the panel
🎯 Key Features
Real-time flight timer and status display
Full flight detail logging (flight number, aircraft, route, ETE)
Multi-language support (English / 简体中文 / 繁體中文)
Save & view mode for distraction-free flight detail display
Native-style bottom bar toggle button with consistent branding
Hotkey K for quick panel access
Hideable button for flexible UI control
Clean, cockpit-optimized dark UI
📝 Notes
This update is fully backward compatible with all previous versions
No changes to core plugin logic — all existing functionality remains 100% intact
UI refinements are designed to match GeoFS native styling for a seamless, immersive experience
