GeoFS FIP Flight Information Plugin v1.2 Update Release Notes
This update focuses on UI unification, feature completion, and experience enhancement, retaining the original logic and layout 100% while delivering practical improvements to better meet flight logging needs.

✨ Update Highlights
1. Unified UI Visual Optimization
Full-panel white text on black background standardization: All buttons, text, and input fields adopt a consistent dark theme, ensuring high visibility in GeoFS flight scenes, reducing eye strain during long flights, and maintaining the original panel structure for zero learning curve.
2. New "Flight Date" Logging Field
Added a Date input field before the flight number, supporting custom entry of flight execution dates to complete flight record dimensions, perfect for airline simulation and flight log management.
Full multi-language synchronization: The field is fully adapted for Simplified Chinese, English, and Traditional Chinese, with no display issues during language switching.
3. Functional Detail Polishing
Reset Button Relocation: Moved the "Reset" button to the left of the "Close" button for more intuitive operation, enabling one-click reset of all flight data with ease.
Default Language Switch: Set English as the default language for international users, while retaining full multi-language switching support (Simplified Chinese / Traditional Chinese).
Redundant Text Removal: Removed "Optional" and other placeholder prompts for a cleaner, more streamlined interface.
4. Experience & Compatibility Upgrade
100% retention of core original features: Flight timer, multi-language switching, K-key panel toggle, floating button control, input conflict prevention, and all other core functions remain fully intact.
One-click Tampermonkey Installation: Compatible with all major browsers via Tampermonkey, ready to use immediately after code paste with no extra configuration.
Input Conflict Fix: Automatically blocks game hotkeys when input fields are focused, eliminating accidental in-flight controls during data entry for smooth operation.

📌 Full Feature List
✅ Flight Information Panel: Supports logging flight date, flight number, aircraft type, departure/arrival airports, and estimated flight time
✅ Flight Timer: One-click start/stop for accurate flight duration tracking
✅ Multi-language Support: Seamless switching between Simplified Chinese, English, and Traditional Chinese
✅ Quick Panel Toggle: Dual control via bottom-left "Geofs FIP" floating button and K-key hotkey
✅ Floating Button Visibility Control: Hide/show the bottom-left button directly from the panel for an unobstructed flight view
✅ Input Conflict Prevention: No game control triggers when typing in input fields for smooth data entry
✅ One-click Reset: Instantly clear all flight data to start a new flight log
✅ One-click Tampermonkey Installation: Full browser compatibility, ready to use out of the box

🚀 Installation & Usage
Copy the full plugin code, create a new script in Tampermonkey, and paste the code
Save the script and refresh the GeoFS page; the plugin will load automatically
Open the panel via the bottom-left "Geofs FIP" button or the K-key, fill in flight details, and start the timer to begin logging

This update delivers comprehensive UI, feature, and experience optimizations without altering the original core logic, making it ideal for daily flight simulation and flight log management.
