## 📌 Features
- ✅ Flight Information Panel: Input flight number, aircraft type, departure/arrival airports, and estimated flight time
- ✅ Flight Timer: One-click start/stop to accurately track your flight duration
- ✅ Multi-language Support: Switch seamlessly between Simplified Chinese, English, and Traditional Chinese
- ✅ Quick Toggle Button: A "Geofs FIP" button in the bottom-left corner to show/hide the panel instantly
- ✅ Button Visibility Control: Hide/show the bottom-left button directly from the panel to keep your flight view unobstructed
- ✅ Key Conflict Fix: No game control triggers when typing in input fields, for smooth data entry
- ✅ One-click Tampermonkey Installation: Compatible with all major browsers via Tampermonkey

## 🚀 Installation
### Method 1: One-click Install via Tampermonkey (Recommended)
1. Install the Tampermonkey extension (supports Chrome, Edge, Firefox, and more)
2. Click the Raw link below, and Tampermonkey will automatically detect and install the script:
   - [Click to Install](https://github.com/Geo-CA/Geofs-Flight-information-plugin/raw/main/main.js)
3. Refresh the GeoFS game page, and the plugin will be active immediately

### Method 2: Manual Installation
1. Copy the full code from `main.js` in this repository
2. Open Tampermonkey → Add a new script
3. Paste the code → Save → Refresh the GeoFS page

---

## 🎮 How to Use
1. Open GeoFS, and the "Geofs FIP" button will appear in the bottom-left corner
2. Click the button to open the flight information panel
3. Fill in your flight details, then click Start to begin the timer
4. After your flight, click Stop to complete the flight log
5. Use the Hide Button option to hide the bottom-left button, and click again to restore it
6. Press the K key to quickly toggle the panel (disabled when input fields are active to avoid conflicts)
