GeoFS FIP Flight Information Plugin v1.4
 
Update Release Notes ✈️
 
 
Update Overview
 
This v1.4 update delivers pixel-perfect UI refinement and consistent branding for the GeoFS FIP plugin, fully aligning the bottom bar toggle button with native GeoFS styling while preserving 100% of core functionality. It creates a seamless, integrated cockpit experience for virtual pilots.
 
 
✨ What's New in v1.4
 
- Branded Bottom Bar Toggle Button
- Renamed button to  Geofs FIP  for accurate, consistent branding
- 1:1 visual match to native GeoFS UI (font size, weight, alignment, spacing)
- One-click panel toggle directly from the in-game bottom bar
- Native-style hover effects and smooth interaction
- Precision UI Alignment
- Fixed text centering for the bottom bar button
- Refined button sizing to perfectly match adjacent native controls
- Zero layout shifts, fully integrated with the cockpit interface
- Core Functionality Preserved
- Real-time flight timer and status tracking
- Full flight detail logging (flight number, aircraft, route, ETE)
- Save & view mode for clean flight detail display
- Hotkey  K  for quick panel toggle
- Full multi-language support (English / 简体中文 / 繁體中文)
- Hideable button for flexible UI control
 
 
 
🛠️ Installation
 
1. Install a userscript manager (Tampermonkey, Violentmonkey, etc.)
2. Click Install on the script page
3. Launch GeoFS — the FIP panel loads automatically in the top-left corner
4. Use the bottom bar  Geofs FIP  button or hotkey  K  to toggle the panel
 
 
 
🎯 Key Features
 
- Real-time flight timer and status display
- Full flight detail logging (flight number, aircraft, route, ETE)
